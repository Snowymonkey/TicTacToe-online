# TicTacToe-Online

TicTacToe-Online is a simple online project created using Python and HTML, which, as the name states, allows for online TicTacToe to be played.

## Features
- Online 2-player TicTacToe
- Separate Game Rooms
- Session-based verification of players

## Installation

```bash

git clone https://github.com/Snowymonkey/TicTacToe-online.git
cd project-name
pip install -r requirements.txt

```
## How to play
python app.py
Go to the locally-hosted website/create (i.e. http://127.0.0.1/create)
Click to join the game
